== Description ==
Ce plugin personnalisable vous permet d'afficher les réseaux sociaux pour partager votre article.  Lorsque le plugin est activé un bloc apparaîtra après le contenu du message avec les liens de partage de différents réseaux sociaux. On peut via l'admin personnaliser l'affichage.

== Installation ==
1. Ajouter le répertoire 'mesreseauxsociaux-plugin' dans le répertoire '/wp-content/plugins/'.
1. Activer le plugin via le menu 'Plugins' dans WordPress.
1. Aller à 'Options->Mes Réseaux Sociaux' dans l'interface d'administration pour sélectionner vos options.

== CSS ==
La feuille de style CSS pour le plugin ce trouve aussi dans ce même répertoire.
