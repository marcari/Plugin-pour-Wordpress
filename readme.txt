== Description ==
Ce plugin personnalisable vous permet d'afficher les reseaux sociaux pour partager votre article.  Lorsque le plugin est active un bloc appara”tra apres le contenu du message avec les liens de partage de differents reseaux sociaux. On peut via l'admin personnaliser l'affichage.

== Installation ==
1. Ajouter le repertoire 'mesreseauxsociaux-plugin' dans le repertoire '/wp-content/plugins/'.
1. Activer le plugin via le menu 'Plugins' dans WordPress.
1. Aller ˆ 'Options->Mes Reseaux Sociaux' dans l'interface d'administration pour selectionner vos options.
